CSS (Cascading Style Sheets):

De qué trata:

  • es un lenguaje de diseño gráfico que se utiliza para definir y crear la presentación visual de documentos estructurados escritos en HTML o XML. A continuación, se detallan los aspectos más relevantes de CSS:
  • CSS se encarga de la apariencia y el formato de las páginas web.
  • Separa el contenido (HTML) de su presentación visual, lo que facilita la gestión y el mantenimiento del diseño.
  • Permite aplicar estilos de manera coherente a múltiples páginas, lo que mejora la consistencia del diseño en todo un sitio web.

Cómo funciona:

  1. Se aplica mediante reglas que asocian propiedades de estilo a elementos HTML.
  2. Utiliza selectores para identificar los elementos a los que se aplicarán los estilos, como etiquetas, clases y IDs.
  3. Las reglas pueden ser aplicadas de las siguientes maneras:
    • - En línea, dentro de las etiquetas HTML.
    • - En el encabezado del documento, utilizando la etiqueta <style>.
    • - En archivos externos, enlazados mediante la etiqueta <link>.

Qué se puede hacer con CSS:

  • Definir colores, fuentes, tamaños y espaciados de texto.
  • Crear diseños responsivos que se adapten a diferentes tamaños de pantalla mediante técnicas como media queries.
  • Animar elementos de la página con transiciones y transformaciones.
  • Controlar la disposición de los elementos en la página (layout) utilizando técnicas como Flexbox y Grid.
  • Aplicar efectos visuales como sombras, gradientes, transparencias o transformaciones.

Función de CSS en una página web:

  • Mejora la apariencia y la experiencia del usuario, haciendo que las páginas sean más atractivas y fáciles de navegar.
  • Permite una mayor consistencia en el diseño a través de múltiples páginas, lo que refuerza la identidad visual del sitio.
  • Facilita el mantenimiento y la actualización del diseño del sitio, ya que los cambios en el CSS se reflejan en todas las páginas vinculadas.
  • Optimiza el rendimiento al separar el estilo del contenido, permitiendo que los navegadores procesen el HTML y el CSS de manera más eficiente.

Propiedades personalizadas (variables CSS):

  • Sintaxis: --nombre-variable: valor;
  • Uso: var(--nombre-variable) para aplicar el valor de la variable.
  • Scope: Pueden ser globales (definidas en :root) o locales a un selector específico.
  • Ejemplo: 
            :root {
          --color-primario: #3498db;
        }
        .boton {
          background-color: var(--color-primario);
        }

Funciones CSS:

  • calc(): Realiza cálculos matemáticos. Ejemplo: width: calc(100% - 20px);
  • clamp(): Limita un valor entre un mínimo y máximo. Ejemplo: font-size: clamp(1rem, 2.5vw, 2rem);
  • min(), max(): Selecciona el valor mínimo o máximo entre varios valores.
  • attr(): Obtiene el valor de un atributo HTML, útil para generar contenido dinámico.

Heurística de cascada y especificidad:

  • Orden de importancia: !important > Estilos en línea > ID > Clase/Atributo/Pseudoclase > Elemento / Pseudoelemento
  • Cálculo de especificidad: (0,0,0,0) representa la especificidad más baja.
  • Estilos en línea: 1,0,0,0 tienen la mayor especificidad.
  • Cada ID: 0,1,0,0 incrementa la especificidad.
  • Cada clase, atributo o pseudoclase: 0,0,1,0 añade un nivel de especificidad.
  • Cada elemento o pseudoelemento: 0,0,0,1 tiene la especificidad más baja.

Layout avanzado con Grid:

  • Grid implícito vs explícito: el primero se genera automáticamente, mientras que el segundo se define manualmente.
  • Unidades fr y minmax(): permiten crear diseños flexibles y adaptativos.
  • Áreas de grid y naming lines: facilitan la organización y el diseño de layouts complejos.
  • Alineamiento y justificación de ítems: controlan la distribución de los elementos dentro del grid.
  • Ejemplo: 
                .container {
              display: grid;
              grid-template-columns: repeat(3, 1fr);
              grid-gap: 10px;
            }
            .item1 {
              grid-column: span 2;
            }

Conclusión:

CSS es una herramienta fundamental en el desarrollo web moderno, ya que permite a los diseñadores y desarrolladores crear interfaces atractivas y funcionales. Su capacidad para separar el contenido de la presentación no solo mejora la organización del código, sino que también facilita el mantenimiento y la escalabilidad de los sitios web. Con el uso de propiedades personalizadas, funciones avanzadas y técnicas de layout como Flexbox y Grid, CSS ha evolucionado para ofrecer un control preciso sobre el diseño y la experiencia del usuario. A medida que la web continúa evolucionando, el dominio de CSS se vuelve cada vez más esencial para cualquier profesional del desarrollo web.
