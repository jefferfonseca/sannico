JS (Java Script):

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que hace que
las páginas web sean dinámicas e interactivas.

¿Para qué sirve?

  • Crea efectos visuales (animaciones, transiciones).
  • Valida datos en formularios.
  • Construye aplicaciones web completas (juegos, apps).
  • Funciona en el lado del servidor (con Node.js).
  • Se usa en desarrollo móvil (React Native).

¿Cómo funciona?

  • Interpretado: El código se ejecuta línea por línea.
  • Cliente: Se ejecuta en tu navegador.
  • Basado en eventos: Reacciona a acciones del usuario (clics, etc.).
  • Orientado a objetos: Organiza el código en objetos.

Conceptos básicos

  • Variables: Almacenan datos (números, texto, etc.).
  • Funciones: Agrupan código reutilizable.
  • Objetos: Representan entidades con propiedades y métodos.
  • DOM: Representa la estructura de un documento HTML.

¿Dónde se usa?

  • Front-end: Crea la interfaz de usuario.
  • Back-end: Construye servidores y aplicaciones.
  • Desarrollo móvil: Crea apps para móviles.
  • Juegos: Desarrolla juegos sencillos y complejos.
  • IoT: Controla dispositivos conectados.

Ventajas

  • Popular: Muy utilizado y con gran comunidad.
  • Versátil: Se puede usar en muchos contextos.
  • Fácil de aprender: Tiene una sintaxis sencilla.

Frameworks populares

  • React: Para crear interfaces de usuario.
  • Angular: Otro framework para interfaces de usuario.
  • Vue: Un framework más ligero y fácil de aprender.
